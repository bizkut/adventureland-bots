# Product Guide

## Project Name
Adventure.land Bots

## Vision
A robust, strategy-pattern based botting framework for Adventure.land to automate gameplay, farming, and events.

## Target Users
Adventure.land players who want to automate their characters using the ALClient library.

## Core Features
- Multi-character coordination (Warriors, Mages, Priests, etc.)
- Strategy Pattern architecture for modular behavior
- Automated farming, banking, and merchanting
- Support for game events (Halloween, Christmas, etc.)
- Complex dungeon automation (Crypt, etc.)

## Success Metrics
- Reliable 24/7 operation without crashes
- Efficient gold and XP farming rates
- Successful completion of complex dungeons
- Clean interactions with other players (no kill stealing)
