# Product Guidelines

## Tone & Voice
Technical, clean, and helpful. Code should be self-documenting where possible.

## Design Principles
- **Modularity**: Use the Strategy and Strategist patterns to isolate behaviors.
- **Reliability**: Bots should recover from disconnects and errors gracefully.
- **Efficiency**: Optimize for server limits and gameplay efficiency (ping compensation).

## Accessibility
N/A - This is a code-based automation tool.
